# GST Invoice Format in India — Complete Guide 2024

The GST invoice format in India is prescribed under the Central Goods and Services Tax (CGST) Rules, 2017. Every GST-registered business must issue invoices in the correct format to claim input tax credit (ITC) and remain compliant with tax authorities.

## Types of GST Invoices in India

Under the GST law, there are different types of invoices based on the nature of supply:

### 1. Tax Invoice
Issued by a **GST-registered supplier** for taxable supplies. This is the most common type of GST invoice used in B2B and B2C transactions.

### 2. Bill of Supply
Issued when the supplier is **not charging GST** — for example, for exempt goods/services or when the supplier is under the composition scheme.

### 3. Receipt Voucher
Issued when **advance payment** is received before the supply of goods or services.

### 4. Refund Voucher
Issued when a **refund is given** against an advance payment received.

### 5. Delivery Challan
Issued for **transportation of goods** that do not involve a sale (e.g., job work, branch transfers).

## Mandatory Fields in a GST Tax Invoice

As per Rule 46 of CGST Rules, a tax invoice must contain:

| Field | Description |
|---|---|
| Supplier Name | Legal name of the registered person |
| Supplier Address | Registered address |
| Supplier GSTIN | 15-digit GST Identification Number |
| Invoice Number | Consecutive unique number |
| Invoice Date | Date of issue |
| Buyer Name | Name of the recipient |
| Buyer GSTIN | Required for B2B transactions |
| Place of Supply | State where supply takes place |
| HSN/SAC Code | Harmonized System of Nomenclature / Service Accounting Code |
| Description | Description of goods or services |
| Quantity | Quantity and unit of measurement |
| Value | Taxable value of supply |
| GST Rate | Applicable GST rate |
| Tax Amount | CGST + SGST (intrastate) or IGST (interstate) separately |
| Total Value | Total amount including GST |
| Signature | Digital or physical signature of supplier |

## CGST and SGST vs IGST — When to Use?

This is one of the most common confusions for new GST filers:

- **CGST + SGST** → When the **supplier and buyer are in the same state** (intrastate transaction)
- **IGST** → When the **supplier and buyer are in different states** (interstate transaction)

### Example:
- A seller in Maharashtra sells to a buyer in Maharashtra → Charge **CGST (9%) + SGST (9%)** for 18% GST
- A seller in Maharashtra sells to a buyer in Delhi → Charge **IGST (18%)**

## HSN Code Requirements

The requirement to mention HSN (Harmonized System of Nomenclature) codes depends on the supplier's turnover:

| Annual Turnover | HSN Digits Required |
|---|---|
| Up to ₹5 crore | 4-digit HSN code |
| Above ₹5 crore | 6-digit HSN code |
| Mandatory (notified items) | 8-digit HSN code |

For **services**, use the **SAC (Service Accounting Code)** instead of HSN.

## GST Invoice Numbering Rules

- Invoice numbers must be **consecutive** (e.g., INV/2024-25/001)
- Numbers should be **unique** for each financial year
- You can use **alphabets, numbers, or special characters** (max 16 characters)
- Invoice series can be reset at the start of each financial year

## Frequently Asked Questions

**Q: Is digital signature mandatory on a GST invoice?**
A: For e-invoices (mandatory for businesses above ₹5 crore turnover), digital signature is required. For other invoices, a physical or scanned signature is acceptable.

**Q: Can I issue a GST invoice without GSTIN?**
A: If you are an unregistered business, you cannot issue a "Tax Invoice." Instead, issue a simple bill/receipt. GST registration is mandatory only if your turnover exceeds ₹20 lakh (₹10 lakh for special category states).

**Q: How many copies of a GST invoice are required?**
A: For **goods**, issue 3 copies (Original for buyer, Duplicate for transporter, Triplicate for supplier). For **services**, 2 copies suffice.
